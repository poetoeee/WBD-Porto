3D Catalog Model Viewer

This project was built to address the need for an interactive and easy-to-use 3D model viewer and catalogs. It allows users to upload, view, and annotate models directly in the browser.

  • Complete annotation system with CRUD with drag and drob feature.

Rush Hour Puzzle Solver

This program, developed in Java using the Swing toolkit for the GUI, implements various pathfinding algorithms to solve the Rush Hour puzzle. The core challenge is to maneuver a primary vehicle out of a gridlocked board through a designated exit point in the minimum number of moves.

Key Features

  • Implements uninformed search (UCS, IDS) and informed search (A*, IDA*) algorithms.
  • Allows user to select different search strategies and heuristic functions.
  • Features multiple heuristics like "Blocking Pieces" and "Manhattan Distance".

Tech Stack

Java Swing
